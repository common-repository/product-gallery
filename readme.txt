Responsive Product Gallery

== Description ==
Product Category & Product Gallery Viewer with Short Description View (optional), Product Page
Widget for Sidebar Product View & Category View

== Installation ==
1. Upload "product-gallery" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. use [lightbox_product_gallery per_page="9" category="slug" desc="true"] short code in your product page
1. use [category_gallery width="215" background="#fff"] short code in your category view page
